# CLICAL

## Мета проєкту
Створити швидку та компактну програму калькулятор із *CLI* інтерфейсом.

## Розширення / Бібліотеки
+ `pytest`: для проведення *init*-тестів.
+ `argparse`: для написання *CLI* логіки.

## Інтерфейс
Плаваючі аргументи:
- `-v, --version`: версія програми.
- `-f, --first-number [INT]`: перше число.
- `-s, --second-number [INT]`: друге число.
Статичні аргументи:
- `func, [add, sub, mul, div]`: арифметичний оператор, статичний аргумент.

```bash
# { clical [func] -f [FIRST NUMBER (int)] -s [SECOND NUMBER (INT)] }
## додавання
clical add -f 10 -s 5
15
## віднімання
clical sub -f 10 -s 5
5
## множення
clical mul -f 10 -s 2
20
## ділення
clical div -f 10 -s 2
5
```
